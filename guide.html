How to Use Lochord

From install to your first DAP-ready playlist in 5 minutes.

1

Install

Download the installer for your OS from the Download page and run it.

  • Windows: Run .exe installer or .msi package. No admin rights needed for the NSIS installer.
  • macOS: Open the .dmg, drag Lochord to Applications.
  • Linux: Use .AppImage (no install needed), .deb, or .rpm.
Windows note: SmartScreen may warn about an unsigned app. Click "More info → Run anyway" to proceed.
2

Choose Music Folder

On first launch, Lochord shows an onboarding screen asking you to select your music root folder.

Lochord
🎵
Welcome to Lochord
Select the root folder of your music library to get started.
📂 Choose Music Folder

Select the top-level folder that contains all your music (e.g. D:\Music or /Users/you/Music). Lochord will create a Playlists/ subfolder here automatically.

DAP tip: Use the same folder structure on your PC as on the DAP's SD card root. This keeps relative paths consistent.
3

Browse Your Library

After scanning, the left panel shows your playlists and the bottom panel shows your folder tree.

  • Click a folder to expand it and see tracks.
  • Use the search bar to filter by filename or tag.
  • Press F5 to rescan if you added new music.
4

Create a Playlist

Click the ＋ New Playlist button in the left panel, then drag tracks from the library into the playlist.

  • Drag a single track from the library onto the playlist panel.
  • Select multiple tracks with Shift+click or Ctrl+click, then drag them all at once.
  • Reorder tracks by dragging within the playlist.
  • Press Ctrl+S to save, or enable auto-save in Settings.
Output format: Playlists are saved as UTF-8 M3U8 files in Music/Playlists/. Paths inside the file are relative (e.g. ../Artist/01.flac), so they resolve correctly on any device.
5

Edit Metadata (Optional)

Right-click any track (or select multiple and right-click) to open the metadata editor. Edit ID3 tags directly without a separate app.

Metadata Editor — Opening.flac
  • Tags are written directly to the audio file (FLAC VORBIS_COMMENT or MP3 ID3v2).
  • Cover art can be previewed and replaced from the editor.
  • Select multiple tracks to batch-edit shared fields.
6

Deploy to Your DAP

Once your playlists are saved, transfer your music to the DAP's SD card, preserving the folder structure.

Astell&Kern · Shanling · HiBy…

The folder structure on your SD card should mirror your PC:

Music/                     ← SD card root
├── Playlists/
│   ├── Favorites.m3u8  ← relative paths inside
│   └── Work BGM.m3u8
├── Artist A - Album (2024) [FLAC]/
│   └── 01. Track.flac
└── Artist B - Album (2023) [MP3]/
  • On the DAP, go to Playlist or File Browser and open any .m3u8 file.
  • Relative paths like ../Artist/01.flac resolve automatically from the Playlists/ folder.
  • Works with Astell&Kern, Activo, Shanling, HiBy, FiiO, Sony Walkman, Cayin, iBasso and any other M3U8-capable DAP.
Pro tip: If you only update playlists (not music files), you only need to copy the Playlists/ folder to the SD card — much faster.
🎉

You're all set!

Your playlists are ready to load on your DAP. Enjoy your music.
