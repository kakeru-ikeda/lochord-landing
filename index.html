Everything you need to manage your music

Lochord gives you full control over your local music collection without any cloud subscription.

📚

Library Scanner

Recursively scans your music directory and displays it as a clean folder tree. Supports FLAC, MP3, AAC, WAV, M4A, OGG, and OPUS.

🎛️

Drag & Drop Playlists

Create M3U8 playlists by dragging tracks from your library. Reorder songs, remove tracks, and save with one click.

🏷️

Metadata Editor

Edit ID3 tags for title, artist, album, genre, year, BPM, lyrics and more. Preview and swap cover art. Batch‑edit multiple tracks at once.

🌗

Dark / Light Theme

Ships with a gorgeous dark theme and a clean light mode. Follows your system preference automatically.

🌍

Multi-language

Full UI support for English, Japanese (日本語), Korean (한국어), and Chinese (中文). More languages welcome.

💻

Cross-Platform

Native desktop app built with Tauri + Rust — runs on Windows, macOS, and Linux with a tiny install footprint.
